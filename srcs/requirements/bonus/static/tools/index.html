MINISHELL

42school's minishell project is a C exercise in building a basic shell. It teaches:

  • CLI: Prompting, parsing, and executing user commands.
  • Built-ins: Implementing commands like cd, echo, and exit.
  • Environment: Managing environment variables.
  • Processes: Forking and waiting for child processes.
  • Redirection/Piping: Redirecting input/output and chaining commands.
It's a great project to learn C programming, parsing, memory management, and error handling.

INCEPTION

The 42 Inception Project is a stepping stone for École 42 students to learn Docker and system administration. They build a multi-container application (often WordPress, MySQL, Nginx, and PHP-FPM) to understand:

  • Creating Docker images
  • Managing containers
  • Deploying multi-container applications.

WEBSERV

Webserv at 42school challenges you to build a basic web server in C++. It covers:

  • HTTP Protocol: Understanding and implementing core functionalities like GET requests and serving static files.
  • Networking: Utilizing sockets for network communication to handle client connections.
  • Concurrency: Managing multiple clients simultaneously using techniques like select().
  • CGI (Common Gateway Interface): Potentially handling dynamic content by integrating with CGI scripts.
    • It's a project to learn C++ networking, socket programming, HTTP, and basic web server functionality.
